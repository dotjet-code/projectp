# Figma プロンプト 04 — インパクト紙面（結果発表 / 月末特番告知 / シェア画像）

**前提**: `01-style-guide.md` に準拠。この 04 は「迫力とキャッチーが最大値」の特別紙面群。通常ページ (02, 03) とは別格の演出予算を使う。

ここは ProjectP というブランドを**一撃で刻む**ための場所。訪問者が SNS でシェアしたくなる、保存してスクショしたくなる、話題にしたくなる紙面を作る。

---

## 共通ルール

- 参照: **横尾忠則・寺山修司の昭和アングラポスター**、**G1 CLIMAX 新日本プロレス興行ポスター**、**映画のキービジュアル**（特に深作欣二・三池崇史系の煽りポスター）
- 全てのページにシグネチャ露出: 篆刻 P + 出走表マーク + PASS LINE マークのどれかが必ず入る
- ポスター見出し (120–240pt) を**臆せず使う**。紙面の 30–50% をタイポグラフィが支配
- 写真は**縁を荒く切り抜く**（鋸歯マスク）。なめらかな PNG 切抜きは禁止
- 紙吹雪・インク飛散・版ズレは**ここでは派手に**使ってよい
- 保存/スクショ/シェアを前提に、**主要情報は画面中央 60% に収める**（トリミング耐性）

## A. 月末特番告知ポスター（`/stage/finale-poster`）

月末特番の 7 日前から節終了まで、トップページに誘導されて到達する専用ポスターページ。興行ポスターを Web に落とす。

### レイアウト

縦長 A2 比率のポスターをブラウザ中央に配置 (`3:4`、デスクトップで `900×1200px`)。左右は紙面地色で埋め、余白に**出走表マーク + 手書きコメント**を散らす。

### 要素構成（上から）

1. **上部大帯** — `--signal-red` ベタ、高さ `120px`。白抜きで「**月末特番**」貂明朝 B 96pt + 右に「**第 12 節 決着**」24pt
2. **ポスター見出し 3 段**:
   - 「**激闘、**」240pt、黒ベタ、画面中央
   - 「**最終節。**」240pt、右 80px オフセット、3 版ズレ（黒 + 赤 + 青）
   - 背景にうっすら「**FINALE**」Unbounded Black 360pt、白 10% opacity で全幅
3. **大判被写体パネル**: 上位候補 3 名の縦長デュオトーン写真を 1:1:1 で横並び、各に号艇色の斜め帯とゼッケン彫込み。各被写体の下にゼッケン + 氏名 + 前節順位
4. **中央朱肉判子**「**決**」`--signal-red` 200pt、被写体パネルと一部重ねて −6° で押す
5. **興行情報帯**（G1 ポスター直参照）:
   - 横罫線で 3 段組:
     - 日付: `2026.04.30` Space Grotesk Bold 72pt
     - 時刻: `21:00〜` 32pt
     - 配信: YouTube / X / TikTok のテキストロゴ列記（アイコンではなく文字）
6. **下部大帯** — 斜め黄黒ストライプ `40px`、その下に「**総視聴 X 万、総投票 Y 万、総バズ Z 億。**」の一行サブコピー、tabular-nums 24pt
7. **左下手書きコメント**「※雨天決行 ／ 観覧無料 ／ 配信あり」（Klee One 赤）
8. **右下篆刻 P** 大きめ `80×80`
9. **フッタ**: 出走表マーク + PASS LINE マーク + 採点規程リンク

### 入場モーション

1. 初回: 画面真っ黒 → 篆刻 P がドン (400ms)
2. 上部赤帯がインク流し込みで左から右へ (600ms)
3. ポスター見出し 3 段が順に（1 段目スライドイン → 2 段目 → 3 段目）各 200ms
4. 被写体 3 パネルが下からスライドアップ 400ms
5. 中央「**決**」判子が 200ms で押される
6. 紙吹雪 (赤 + 黒 + 黄、各 10 枚) が 2 秒撒かれる
7. 以降静止、ユーザーは保存 / シェアに向かえる

## B. 結果発表紙面（`/results/[stageId]`）

節決着後の紙面。興行結果 × スポーツ新聞一面 × 壁新聞の迫力を融合。

### 全体構成（縦スクロール）

#### B-1. 速報帯（ページ最上部、全幅）

- 高さ `80px`、`--signal-red` ベタ
- 左にスタンプ「**速報**」白抜き 48pt
- 中央「**第 12 節、決着。**」貂明朝 B 48pt 白
- 右に日時（`2026.04.30 21:47 JST` Space Grotesk Bold）

#### B-2. 勝者紙面（画面 100vh 級）

- 左 1/2: **勝者の大判ポートレート全幅** (縦 `800px`+)、デュオトーン + 号艇 1 号(白)縁で荒切抜き
  - 写真内にゼッケン彫込み 320pt
  - 写真上に横尾忠則的な斜め赤帯が 2 本貫通
- 右 1/2: 多段見出し
  - 赤スタンプ `[優勝]` 24pt
  - 「**〇〇、**」貂明朝 B 144pt
  - 「**首位。**」貂明朝 B 200pt、右オフセット、3 版ズレ
  - サブ「12 節、2,081 ポイント。最終 3 日で追い上げ、2 位 〇〇 を 47pt 差で退ける。」（32pt）
  - その下に勝者インタビュー本文（長文、16pt 和文、段落は新聞カラム式）
- 背景に「**WINNER**」Unbounded Black 400pt、2% opacity、全幅
- 右下に朱肉判子「**勝**」120pt

#### B-3. 全 12 位の結果表

新聞の競馬結果欄を完全に模した**馬柱式**テーブル。

- 上から下へ 1〜12 位、12 行
- 各行のカラム:
  - 順位（Space Grotesk Bold 64pt、1-3 位は赤、4-6 位は黒、7-12 位は灰）
  - ゼッケン + 号艇プレート
  - 氏名（貂明朝 B 24pt）
  - 最終ポイント（Space Grotesk Bold 32pt）
  - 前節比 (▲/▼/−)
  - バズ / 配信 / 収支の 3 数字、ミニスコアバー付き
  - 「敗者復活通過」バッジ（該当者のみ、`--signal-yellow` ベタ）
- 6 位 / 7 位の間に**太い PASS LINE 帯** 赤 `6px` + ラベル「— PASS LINE —」
- 表全体は黒 1px ケイ線のみで区切る。カード化しない

#### B-4. 名場面紙面

「今節の名場面」を 3 コマで横並びに紙面化。各コマ:
- 中判写真（デュオトーン、`300×300`、荒切抜き）
- 見出し（貂明朝 B 32pt）「**3 日目、塩見きらが首位を奪取。**」
- 本文 2 行 + 発生時刻
- 右下に小さな判子（「差し」「逃げ」「捲り」いずれか）

#### B-5. 敗者復活結果

敗者復活戦の勝者（PIT から PLAYER 昇格）を独立帯で紹介:
- 帯全幅 `--signal-yellow` ベタ、高さ `240px`
- 左: 勝者ゼッケン + 氏名 貂明朝 B 96pt
- 中央: 「**PIT から PLAYER へ。**」48pt
- 右: 判子「**逆転**」赤 120pt −8°
- 背景に薄く「REVIVAL」Unbounded 240pt

#### B-6. 予想王発表

節 6 連単予想で最も的中した予想者 TOP 3:
- ミニ表彰台ではなく、新聞の**読者投稿欄**風に 3 列組
- 予想者ハンドル + 的中数 + 総投票数 + 獲得バッジ
- 上位には判子「**的中**」

#### B-7. 次節告知

下部に**次節の出走表**（前節結果から自動生成された号艇 1-6 のアサイン）を馬柱式で掲載:
- 「**第 13 節 出走表**」見出し
- 1 号艇（新首位）から 6 号艇まで 6 名 + PIT 6 名
- 「**開幕 2026.05.01 0:00**」の赤ベタ帯
- CTA「**予想する →**」`--signal-red` ボタン

#### B-8. フッタ

- 篆刻 P + 出走表マーク + PASS LINE マーク
- 「**本節結果は採点規程に基づく →**」リンク
- 更新時刻

### 入場モーション

- ページ到達時、上部赤速報帯がインク流し込みで展開 (600ms)
- 勝者紙面の見出し 3 段が順に登場 (各 300ms)
- 「**勝**」判子が押される (200ms)
- 紙吹雪 3 秒、赤 + 金色 + 黒 (結果発表のみ金色許可)
- 以降セクション毎にフェードイン

## C. 敗者復活戦告知（`/stage/revival-poster`）

結果発表の前段階、敗者復活戦の開催前に掲出されるポスター単独ページ。B の月末特番告知よりも**アンダーグラウンド寄り**のトーン。

### レイアウト

縦長 `900×1200px`、黒地。

### 要素

1. 上部帯 `--signal-yellow` ベタ、黒文字「**敗者復活戦**」A1 ゴシック 96pt
2. 大判被写体: PIT 6 名のシルエットを**ステンシル風**に画面下半分いっぱいに並べる（日比野克彦参照）
3. ポスター見出し:
   - 「**PIT、**」貂明朝 B 200pt 白
   - 「**上げろ。**」貂明朝 B 240pt、3 版ズレ（白 + 赤 + 青）
   - 背景に「REVIVAL」Unbounded 320pt、白 5%
4. 判子「**待て**」右上、赤 160pt
5. 興行情報帯 (G 1 参照):
   - 日付・時刻・配信先
6. 「**7 位以下の 6 名から、1 名だけが PLAYER へ。**」30pt 白、下部
7. CTA「**投票する →**」`--signal-red` ベタ、押すと `/live/vote` に遷移
8. 下部に手書き赤コメント「※現地投票あり／配信視聴可／投票はログイン必須」

## D. SNS シェア画像テンプレート

保存・拡散されることを前提にした、事件単位の画像を自動生成。OG image + Twitter カード + Instagram 縦型の 3 比率。

### D-1. 正方 (Instagram / OG サムネ, `1080×1080`)

- 全幅赤ベタ上部帯「**速報**」白抜き
- 中央に事件の主役メンバー デュオトーン大判
- 右にポスター見出し 2 段 (「首位、」「陥落。」等)
- 下に出走表マーク + 篆刻 P
- 角に日付スタンプ

### D-2. 横長 (Twitter / OG, `1200×630`)

- 左 40%: 被写体縦長
- 右 60%: 見出し + サブコピー + ゼッケン + ポイント
- 背景に「PROJECT P」Unbounded 240pt、白 5%
- 右下に篆刻 P

### D-3. 縦長 (Instagram ストーリー, `1080×1920`)

- 上部 20%: 赤帯 + 事件タイプ
- 中部 60%: 大判被写体 + ポスター見出し 3 段
- 下部 20%: 出走表マーク + CTA「詳しく →」+ 篆刻 P

### 生成ロジック

- サーバーで OG 画像を Satori / `@vercel/og` で生成
- テンプレートは 4 種（速報 / 結果 / 告知 / ポスター）
- データ差し込み: 見出し、サブ、被写体 URL、日付、ゼッケン、色
