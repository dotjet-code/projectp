# ビジュアル参照リスト — Figma Make ムードボード用

Figma Make / Figma AI に世界観を理解させるためのリファレンス画像集。下記 URL を開き、**実画像を保存 → Figma 内キャンバスに 1 枚 1 枚貼る**。その上で「このムードで」と指示すると再現率が跳ねる。

優先度高い順に並べる。最初の 8 枚だけ貼れば十分効く。

## Figma Make での使い方（推奨手順）

### Step 1: ムードボード作成
- Figma の新規キャンバスに "Mood Board" フレームを用意
- 上記 A-1 〜 A-8 の画像を保存し、横 4 × 縦 2 で並べて配置
- キャプションは不要、画像だけ

### Step 2: 参照指定プロンプト
Figma Make に投げるプロンプトの冒頭に以下を追加:

```
Reference visual style: The mood board in this file combines
(1) Tadanori Yokoo 1960s-70s posters (red-black silkscreen, mis-registration, photographic cutouts on flat color),
(2) Japanese sports newspaper front pages like Tokyo Sports (bold outlined headlines, period-punctuation taglines),
(3) Saul Bass film posters (flat silhouette geometry),
(4) Peter Saville's Unknown Pleasures cover (architectural minimalism on black).
NOT idol/cute/pastel. NOT gold-on-black luxury. Target: gritty, hot, competitive.
```

### Step 3: スクリーンプロンプト投入
そこから [compact/02-top-page.md](02-top-page.md) の MUST 5 項目をペースト。ムードが前提になるので「大判人物写真」が"モノクロ切抜き+赤ベタ帯"で出力される確率が跳ねる。

### Step 4: 1 項目ずつ追加
- 「右上に朱肉判子を追加。横尾忠則のポスターにあるような赤い正方スタンプ、-6° 傾き、中に『激震』の太い和文」
- 「見出し『陥落。』の背景に同じ文字を赤で 3px 右下にズラして重ねて。横尾忠則の版ズレのように」
- のように、参照作品名を毎回添える

### Step 5: NG 指定を必ず入れる
最後に毎回「**avoid: rounded corners, pastel colors, drop shadows, gradients, icons for rank medals (🥇🥈🥉), circular profile photo crops, cute illustrations**」を添える。これを入れないと AI は無難な方に寄る。

## 参考: Figma Make に画像 URL を直接投げる場合

一部のツールは URL からの画像取得に対応する。その場合は直接リンク可能な画像:

- Wikipedia の公開画像（`/wiki/File:` のサムネイル URL は安定）
- MoMA の作品ページ画像（ページ表示される JPG を右クリック → 画像アドレスをコピー）
- Letterform Archive、Wallpaper* 等の記事画像（CDN 直リンク可）

ただし権利上の問題があるので**個人のムードボード用途に限定**し、生成された成果物の公開時にはリファレンスに依存しない独自ビジュアルになっているかを確認すること。
